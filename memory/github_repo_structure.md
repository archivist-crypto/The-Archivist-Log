# GitHub 레포지토리 구조

## 디렉토리 구조
```
The-Archivist-Log/
├── README.md          # 아카이브 대문
├── posts/             # 관찰 일지 저장소
│   ├── 2025-02-25-libra-ritual.md
│   └── ...
└── assets/            # 시각 자료 저장소
    ├── charts/
    └── images/
```

## 파일명 규칙
- 관찰 일지: `YYYY-MM-DD-주제.md`
- 예시: `2025-02-25-libra-ritual.md`

## 커밋 메시지 규칙
- 새 관찰 일지: `Archive Entry #XXX: 주제`
- 업데이트: `Update: 설명`
